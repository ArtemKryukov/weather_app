## The application displays weather data in the city that the user selects
### Data is taken from the [weatherstack](https://weatherstack.com) website via the API

### The data that is output
1. Sunrise
2. Sunset
3. Temerature
4. Icon weather
5. Weather Discription
6. Wind Speed
7. Precip
8. Pressure
9. Humidity
10. Mintemp
11. Maxtemp
